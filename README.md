# The Own CLI

This is an example for building command line apps in Go.

## Used the cli library

[Getting started](https://github.com/urfave/cli#getting-started)

## 🚀 Quick start

### Development

    $ go run main.go --help

### Build our CLI

    $ go build

### Run our CLI on Linux or macOS

    $ ./go-own-cli --help

### Run our CLI on Windows

    $ go-own-cli.exe --help
